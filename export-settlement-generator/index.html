步骤 1 / 4
协议头
协议第一页最顶部的三行：编号、签订日期、签订地点。日期与地点同时填写中俄文，PDF 会显示为「中文 / Русский」格式。

协议头

协议第一页最顶部的三行：编号、签订日期、签订地点。日期与地点同时填写中俄文，PDF 会显示为「中文 / Русский」格式。

这一节填什么
  1. 协议编号：例如 DLJC-2026-001。
  2. 签订日期：中文 + 俄文各一份。
  3. 签订地点：城市名，中俄各一份。
甲乙双方
填写委托方（甲方）与出口代理方（乙方）信息。中文必填；俄文如不会翻译可暂留空，PDF 中会以下划线占位。

委托方（甲方）信息

甲方 = 委托别人帮忙出口的人/公司，通常是有货物要卖出口的中国公司。中文必填；俄文如不会翻译可暂留空，PDF 中会以下划线占位。

逐项填写指引
  • 公司名称：营业执照上的全称（中、俄各一份）
  • 统一社会信用代码：18 位编码（俄文版可与中文一致）
  • 地址 / 联系人 / 电话 / 邮箱：中、俄各一份

出口代理方（乙方）信息

乙方 = 替甲方做出口代理 / 收外汇的人/公司。统一信用代码、地址、联系人都建议填，便于法律核验。

代理费率、账户与实际控制人
协议核心金额条款。比例字段只填数字，不要带 %。账户信息会写入第 IV 节中俄两版。

代理费率、账户与实际控制人

这是协议的核心金额条款，非常关键。比例字段只填数字，不要带 %。

必填四组数据
  1. 代理费比例（按收汇金额 % 计）+ 单笔最低代理费（CNY）
  2. 代垫资金占用年化（垫付货款时按这个利率计息）
  3. 甲方收款账户：公司名 + 开户行 + 账号
  4. 乙方实际控制人姓名（承担连带担保）
签署日期与商品明细
协议末尾甲乙双方签字旁的日期，以及协议正文末尾的商品表（最多 5 行，可选）。确认后生成 PDF。

签署日期

协议末尾甲乙双方签字旁的日期。填阿拉伯数字即可，俄文段会自动转成 июня 等单词。

批次商品明细（可选）

协议正文末尾有一张商品表（最多 5 行）。如还没确定具体商品，可全部留空，PDF 中表格会以下划线占位。

实时填写预览

这里只展示主要替换字段，最终 PDF 以 Word 模板排版为准。

填写常见疑问

① 协议编号会同时出现在中俄文标题后；② 比例字段只填数字（如 1.2，不要带 %）；③ 俄文公司信息可与中文一致，也可留空；④ 商品明细表最多支持 5 行。

已保留的填写内容

协议编号、签订日期/地点、甲乙双方公司中俄文 + 信用代码 + 地址 + 联系人 + 电话 + 邮箱、代理费率/最低代理费/年化、甲方收款账户、乙方实际控制人、签署日期、货物品名、动态商品明细行（含「+ 添加一行商品」）。
