Payable Prepay Agreement Workflow
步骤 1 / 4
协议头
填写协议第一页顶部的编号、签订日期和签订地点。建议中俄文信息同时填写，便于模板同步替换。

协议头

协议第一页最顶部三行：编号、签订日期、签订地点。建议中俄文同时填写。

这一节填什么
  1. 协议编号：例如 SM-TH01
  2. 签订日期：中文 + 俄文各一份
  3. 签订地点：城市名，中俄各一份
甲乙双方
填写甲方（垫付方）和乙方（垫付委托方）的公司、税号、地址、联系人、电话和邮箱。

甲方（垫付方）信息

甲方 = 提供垫付服务的一方。模板默认是智鏈通達，如实际业务中甲方有变，请全部修改。

甲方必填项
  • 公司名（中/俄各一份）
  • INN / KPP（俄罗斯税号体系）
  • 地址 / 联系人 / 电话 / 邮箱（中俄各一份）

乙方（垫付委托方）信息

乙方 = 找甲方垫付应付账款的进口商。模板默认是 AIRFLOT，必须改成实际客户。

乙方必填项
  • 公司名（中/英 + 俄文各一份）
  • INN / KPP / 统一社会信用代码
  • 地址 / 联系人 / 电话 / 邮箱（中俄各一份）
正文条款与附件
填写正文中的贸易出口商、联系邮箱、附件一费率确认单、附件二服务期及服务费用明细。

正文条款中的可变信息

协议第一条里嵌着贸易出口商公司名和双方联系邮箱，这些都会出现在中俄文正文中。

需要修改的内容
  1. 贸易出口商公司名：乙方与谁签的贸易合同（默认 GUANGDONG OSHUJIAN）
  2. 乙方联系邮箱：正文 1.2 条款中乙方指定的联系邮箱
  3. 甲方联系邮箱：正文 1.2 条款中甲方指定的联系邮箱

附件一：垫付资金占用费率确认单

这是协议核心金额条款。比例字段只填数字（不带 %）。

必填四项
  1. 资金占用年化费率（如 8 表示 8%）
  2. 逾期年化费率（如 18）
  3. 订单编号 & 客户编号
  4. 乙方在中国联络人姓名

附件二：垫付资金占用费确认单

这是结算单，记录服务起止日期与本次结算金额。未发生实际垫付前可留空。

附件二：服务费用明细（可选）

附件二里有一张 3 行费用明细表。点击「+ 添加一行」可增加；不需要时全部留空。

签署与生成
填写协议正文末尾、附件一、附件二共 5 处签字栏的日期与授权代表姓名。确认后生成 PDF。

签署信息

协议正文末尾、附件一、附件二共 5 处签字栏的日期与授权代表姓名。

签字栏填写
  1. 签署日期：年月日，会同步到所有 5 处签字栏
  2. 甲方授权代表姓名：签字栏打印名
  3. 乙方授权代表姓名：签字栏打印名

实时填写预览

这里只展示主要替换字段，最终 PDF 以 Word 模板排版为准。

填写常见疑问

① 比例字段只填数字，不要带 %；② 甲方默认是智鏈通達，如不变可不填；③ 乙方必须改成实际客户公司名；④ 服务期/合计金额留到结算时再填也可以；⑤ 服务费用明细保留原来的添加行方式。

已保留的填写内容

协议编号、签订日期/地点、甲乙双方公司和联系方式、贸易出口商、双方正文邮箱、附件一协议/订单/客户编号、年化和逾期费率、中国联络人、附件二服务期、合计金额、服务费用明细、签署日期和授权代表。
